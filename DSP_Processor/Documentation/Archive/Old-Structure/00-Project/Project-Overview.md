# DSP_Processor - Project Overview

**Date:** January 15, 2026  
**Status:** Active Development - Phase 3.1  
**Language:** Visual Basic .NET  
**Framework:** .NET 8.0

---

## ?? **What is DSP_Processor?**

DSP_Processor is a **real-time audio processing application** with professional-grade DSP capabilities, visualization, and recording features. Built as a learning project and reference implementation for audio DSP in VB.NET.

---

## ? **Key Features**

### **??? Audio Processing**
- Real-time DSP chain with multiple processors
- Gain control with constant-power pan law
- High-pass and low-pass filters (in progress)
- Output mixer with stereo width control
- Zero-latency passthrough mode

### **?? Visualization**
- Real-time FFT spectrum analyzer
- Dual-channel waveform display
- VU meters with peak/RMS display
- Configurable visualization settings

### **??? Recording & Playback**
- Multi-format recording (WAV, MP3, FLAC)
- High-quality audio playback
- File management and organization
- Transport controls

### **?? Professional Features**
- WASAPI exclusive mode support
- Asynchronous logging system
- Flexible audio routing
- Dark-themed professional UI
